
IElectronics.h - Общий абстрактный класс для всей создаваемой техники и её элементов (только заголовочный файл)

InputDevices.h (.cpp) - Заголовочный и cpp файлы устройств ввода. В них содержатся классы "Контроллер" и "Игровая приставка".
			Класс "Контроллер" наследует класс "IElectronics". 
			Класс "Игровая приставка" наследует класс "Контроллер" и, как следствие, класс "IElectronics".

OutputDevices.h(.cpp) - Заголовочный и cpp файлы устройств вывода. В них содержатся классы "Экран" и "Телевизор".
			Класс ""Экран" наследует класс "IElectronics". 
			Класс "Телевизор" наследует класс "Экран" и, как следствие, класс "IElectronics".

PC.h (.cpp)           - Заголовочный и cpp файлы класса "Персональный компьютер". 
			Наследует свойства от всех классов устройств ввода и вывода и, как следствие, класса "IElectronics".

main.cpp 	      - Файл запуска программы с объявлением объектов и выводом информации на экран по требованию пользователя.
